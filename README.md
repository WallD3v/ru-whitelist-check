# Russia Mobile Internet Whitelist Checker

## **RU**
Простой сайт, который проверяет, находится ли заданный IP-адрес в белых списках (точное соответствие IP или попадание в CIDR-диапазоны), которые остаются доступными при ограничении мобильного интернета в РФ.

## **EN**
A simple site that checks if a given IP address is present in the whitelists (exact IPs or CIDR ranges) that remain accessible when mobile internet is restricted in Russia.
